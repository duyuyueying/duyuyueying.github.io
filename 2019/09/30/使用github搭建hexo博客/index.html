使用github搭建hexo博客

######1.安装Node.js和配置好Node.js环境。

######2.安装Git和配置好Git环境。

######3.Github账户注册和新建博客项目。

注：项目必须要遵守格式：账户名.github.io；并且需要勾选Initialize this repository with a README。

在建好的项目右侧有个settings按钮，点击它，向下拉到GitHub Pages，你会看到那边有个网址，表示该项目已经被部署到网络上，能够通过外网来访问它。


######4.安装Hexo。
安装Hexo

检查hexo是否安装成功。


在自己认为合适的地方创建一个文件夹，然后进入到该文件夹，初始化该文件夹，


出现Start blogging with Hexo!即为成功
安装初始化后的所需要的组件

生成静态文件。

开启服务，访问该网址，体验hexo

浏览器打开localhost:4000/出现下图，就成功了

######5.将本地与github联系起来。
设置git的user name 和email
检查是否有.ssh的文件夹

如果没有就生成一个密钥

连续三个或四个回车，得到两个文件：id_rsa和id_rsa.pub
在github上添加公钥public-key:
复制id_ras.pub的内容，在github上添加公钥，直接粘贴，保存即可

查看ssh是否添加成功

######6.将hexo与github联系起来，在文件夹中，找到_config.yml文件，修改repository值

######7.新建一篇博客。

这时候可以在文件夹_posts目录下将会看到已经创建的文件。

######8.生成部署文章

在生成及部署文章前，需要安装一个扩展

部署成功后访问你的地址，就会看到刚刚生成的文章了。
